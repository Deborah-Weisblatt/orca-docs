API Tokens

automatically creates four tokens for each new project. You can create as many additional authentication tokens as you require. The default tokens created for a new project are:

Name Description
badge authentication token to use when integrating in your code repository, for displaying the Tufin badges with your Docker images
CI authentication token to use when integrating in your CI/CD pipeline
deploy authentication token to use when deploying in your cluster
kite authentication token use by the kite pod
