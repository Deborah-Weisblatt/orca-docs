Logging in

The login screen is located at https://securecloud.tufin.io/login or https://securecloud.tufin.io/<>, where <> is your account name.

Once you have specified your account you can log in with your credentials. If you have correctly set up single signon, you will see an additional option to log in via .

How do I Setup for Single Signon?

  1. Log into your account

  2. Create a new app registration with the following properties:

    • App registration name =
    • Redirect URI = https://securecloud.tufin.io/auth/realms/<>/broker/azure-ad/endpoint, where <> is your account name
  3. Add a new client secret to the app registration
  4. Add a token configuration, new groups claim, specifying type security groups

  5. You will need the following details from your account:

    • Directory (tenant) ID
    • Application (client) ID, created in step 2 above
    • Client secret value, created in step 3 above
    • Group claim token value, created in step 4 above
