Install the Connector on

  1. Click on View install script and then click on the copy icon . The script will be copied to the clipboard.
  2. On the server, log in to the console.
  3. Paste the script copied above from to the console and run the script.
  4. After the script has completed running on the server, refresh the screen - .

  5. If the script has been successfully installed, the screen will appear as follows:

Enter the Workflow Details in

  1. Click Edit . The Configure SecureChange Integration window appears.

  2. Enter an existing user name, password and workflow defined in .
  3. Save.

Uninstall the Connector From

  1. Click on View install script and then click on the copy icon . The script will be copied to the clipboard.
  2. On the server, log in to the console.
  3. Paste the script copied above from to the console.
  4. In the script change the text "install-connector" to "uninstall-connector".
  5. Run the script.
