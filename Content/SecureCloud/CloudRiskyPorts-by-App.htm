by Application

The risky ports application view lists all the applications that have one or more assets with one or more high risk ports open.

What Are ?

The main screen section is a list of applications showing the following information:

  • Application name

  • Internet access:

    Exposed-one or more of the application’s assets has access to the internet

    blank-none of the application’s assets has access to the internet

  • Total assets. The total number of assets in the application

  • Affected assets. The number of assets in the application with risky ports

  • Risky ports. A list of all risky ports identified in the application's assets
