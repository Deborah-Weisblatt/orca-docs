This screen displays all clusters containing privileged workloads.

What are privileged workloads?

The main screen section is a list of clusters showing:

  • Cluster name

  • Total namespaces. The number of all namespaces in the cluster

  • Affected namespaces. The number of namespaces in the cluster that contain privileged workloads

  • Host Access. The number of privileged pods

  • Privileged mode. The number of containers with privileged parameter set to true

  • Drill down to see Privileged Workloads by Namespace

    - click on the desired cluster.
