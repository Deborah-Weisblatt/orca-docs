Tokens

API tokens are unique identifiers used by to give other applications access to it. will accept a request from another application only if the request is made with a token that matches that stored by . Some tokens are created for you automatically; others must be created manually.

Tokens have three properties - scope, label and the token value

Token Property Values
Scope

VIEW - can make read-only API requests to: 'https://orca.tufin.io/api/*'

AGENT - used only by the

ALL - can make API requests to all of Orca's endpoints
Label Free format text containing English characters, upper or lower case, spaces and hyphens

Scope / Label Description

all / CI authentication token to use when integrating in your CI/CD pipeline
view / deploy authentication token to use when deploying in your cluster
