Firewalls

You can apply the Orca security policy to firewalls using any of the following methods:

  • On a firewall managed by
  • On a Microsoft Azure firewall

Currently only egress rules are supported.

Each method requires a one-time integration setup, which enables the Apply on firewalls option on the Security Policy screen.

Apply your security policy to the appropriate firewall devices by clicking on the 'apply on firewalls' function. This will submit one or more firewall change tickets to the device. These tickets can then be viewed on the Firewall Tickets Screen.

 

See also:

  • SecureChange integration

 

Steps Required on Microsoft Azure

  1. Create a tag for the asset:

    Key =

    Value = <domain1>.<project1>, <domain2>.<project2>...

  2. Add application rule collection:

    Name = tufin.orca.<domain>.<project>, where

    Action = Allow

    Add a separate rule for every Orca project
