Firewalls

You can apply the Orca security policy to firewalls using any of the following methods:

  • On a firewall managed by
  • On one or more Microsoft Azure firewalls. Currently only egress rules are supported.

Each method requires a one-time integration setup, which enables the Apply on firewalls option on the Security Policy screen.

Apply your security policy to the appropriate firewall devices by clicking on the 'apply on firewalls' function. This will submit one or more firewall change requests or tickets to the device. The status of these tickets can then be viewed on the Firewall Tickets Screen.
