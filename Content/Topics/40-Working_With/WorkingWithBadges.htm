Overview

An badge shows the security score of a single container in real time (the image, not the source code). See Badge Concepts.

Implementing the Badge

Inserting badges into repository readme files is a popular and convenient way of making the available in a prominent position. Just enter the API string with the appropriate parameters.

Format:

![Tufin](https://orca.tufin.io/api/<domain>/<project>/badges/security-score?image=<docker-image-name>:cia-latest&token=<view token>)](https://orca.tufin.io/ui/#/<domain>/<project>/grid/scans?image=<docker-image name>)

Where:

Parameter Source of Value
<domain> > at top of most screens
<project> > at top of most screens
<docker-image-name> Container name as displayed in pod
<view token> > Settings > API Tokens > Token value for [Scope=view Label=badge]

Example:

![Tufin](https://orca.tufin.io/api/MyCompany/MyProject/badges/security-score?image=k8s.gcr.io/event-exporter:05:cia-latest&token=8440d55d-97a9-47c8-928a-20963cf14b14)](https://orca.tufin.io/ui/#/MyCompany/MyProject/grid/scans?image=k8s.gcr.io/event-exporter:05)
