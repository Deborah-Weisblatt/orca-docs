Filtering Displayed Data

Filter by Domain and Project

When you sign up for a account, your username is assigned a domain and project. The Domain and Project are displayed in the top bar of .

If your username is associated with multiple domains, you can select domain you want to display. For each domain, select which specific project you want to view.

Filter by Namespace

lets you filter the data displayed for a specific namespace in your cluster. By default, displays the aggregated data for all namespaces in your cluster. The currently selected filter remains active as you navigate to different windows. To change your namespace filter criteria, select a new filter. To view the aggregated data for all namespaces in your cluster, select all.

To view the data for a specific namespace, select the desired namespace.
