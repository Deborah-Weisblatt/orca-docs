Signing Up

To sign up for a Account

  1. Go to the Tufin Orca homepage and click sign up.

  2. Fill in the requested information and submit the form.

You will receive a follow-up email containing everything you need to get started using :

Email Data Description
Username Username will be based on the email address provided
Domain Domain will be based on the company name provided
Project A project name, based on the project name you provided
Deployment Command Line The bash command string you need run from a terminal window

The deployment command line has the following format:

bash <(curl -s https://orca.tufin.io/api/<domain>/<project>/scripts/deploy-orca -H "Authorization: Bearer <deploy_token>")

where,

    Variable Description
    <domain> Company or organization name
    <project> Project, application, or Kubernetes cluster
    <deploy_token> Authentication token named deploy for the desired project
