This screen displays the topology of a single application, showing its associated nodes, the connections between them and connections with entities outside of the application. The arrows indicate traffic permitted in the VPC, as defined by the VPC security objects and routing tables. To make viewing easier, icons can be moved around using the mouse.

Icons are displayed in red when there is a violation.

Reposition Icons

To make viewing easier, icons can be moved around the screen using the mouse. The default layout is restored whenever the screen is redisplayed.

Display Associated Assets

View the assets associated with an application.

  1. To display all assets, click All Assets at the top of the screen. The Assets Screen will be displayed filtered to show all the application's assets.
  2. To display only risky assets, click on Risky Assets at the top of the screen. The Assets Screen will be displayed filtered to show only assets with violations.

View Node Details
