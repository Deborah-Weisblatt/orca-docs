Applications

View your applications as specified by the tags associated with your assets. You can drill down to view the application graph and to the assets associated with it.

  • Filter the application list
  • Drill down to a selected application
  • Display application issues
  • Display the asset map
  • Drill down to associated assets
  • View node details

Filter the Application List

Applications are represented by cards showing the following information:

Field
Sequence of item in the list
Application Name in the format specified in the App Manager
The number of assets associated with the application

The cloud vendor
Network connectivity - internet connectivity, network peering and/or on-premise network
Warning - indicates a violation

The list can be filtered on the following criteria:

  • Violations
  • Internet connectivity
  • Network peering
  • On-premises connectivity
  • Application name
  • Tags

Display Asset Map

  1. Click on the desired application from the list or double-click it on the [%=local-iris.GlobalMap%]. The topology of the application is displayed including associated assets, nodes outside of the application and the connections between them.

    The application is represented by nodes connected by arrows indicating traffic permitted in the VPC, as defined by the VPC security objects and routing tables.

    Icons are displayed in red when there is a violation.

  2. Node details. Click on a node on the screen to display more information. When a node is clicked, its background becomes shaded to indicate it is currently selected. See View Node Details
  3. Drill down. Double click on an asset to display the Asset Detail Screen. You cannot drill down to nodes that are not application assets.

Display Associated Assets

View the assets associated with an application.

  1. Display the Asset Map for an application.
  2. To display all assets, click All Assets at the top of the screen. The Assets Screen will be displayed filtered to show all the application's assets.
  3. To display only risky assets, click on Risky Assets at the top of the screen. The Assets Screen will be displayed filtered to show only assets with violations.
