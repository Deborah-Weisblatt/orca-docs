Overview

An badge shows the of a single container (the image, not the source code) in real time. See Badge Concepts.

Implementing the Badge

Inserting badges into repository readme files is a popular and convenient way of making the available in a prominent position. Just enter the API string with the appropriate parameters.

Format:

![Tufin](https://orca.tufin.io/api/<domain>/<project>/badges/security-score?image=<pod-image-name>:<image-tag>&token=<view token>)](https://orca.tufin.io/ui/#/<domain>/<project>/grid/scans?image=<pod-image name>:<image-tag>)

Where,

Parameter Where to Find the Value
<domain> > at top of most screens
<project> > at top of most screens
<pod-image-name> Image name as displayed in pod
<image-tag> The tag ID for the appropriate version of the image - should be the last one you created
<view token> > Settings > API Tokens > Token value for [Scope=view Label=badge]

Example:

![Tufin](https://orca.tufin.io/api/MyCompany/MyProject/badges/security-score?image=k8s.gcr.io/event-exporter:05&token=6440d59a-97a9-47c8-92a-20963fc47b11)](https://orca.tufin.io/ui/#/MyCompany/MyProject/grid/scans?image=k8s.gcr.io/event-exporter:05)

Where,

Parameter Value
<domain> MyCompany
<project> MyProject
<pod-image-name> k8s.gcr.io/event-exporter
<image-tag> 05
<view token> 6440d59a-97a9-47c8-92a-20963fc47b11
