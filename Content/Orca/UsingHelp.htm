Using the Help Center

Screenshot Thumbnails

Many screenshots are initially shown as thumbnail pictures to reduce page size and make it easier for the reader to quickly understand the structure of the help page. Thumbnails are identified by the dotted frame surrounding the picture. Click on a thumbnail to expand the picture to full size. Click again minimize it.

Text Emphasis

Different font characteristics are used to distinguish between elements:

  • Regular text
  • Screen object - either a complete screen or a component of it. Includes field names, screen names, buttons, screen panels and other components. This is used when describing an action on the screen object (like 'click Continue') or referring to it from another place, However, this is generally not used when describing the item itself, such as in a help document dedicated to it.
  • Field value or status (e.g. Learn Mode, Enforce Mode, On, Off)

Screen Theme Colors

At any given time, the application screens will follow one of two color themes, depending on the Policy Mode currently selected. The help screenshot may therefore appear with a different color theme than the one you currently see in the application.
