Getting Help in the Application

The ? help icon is displayed at the top of every screen; click on it to display the Help Menu. Options:

  • Help center - takes you to the start page of the online help

  • What can I do on this page? - takes you to the help topic specific to the current screen
  • What's new - takes you to the help topic that lists changes in the product
  • Contact us - opens a new email message for you to send to support

Refreshing the Page

While you are working in the application, the data displayed on the screen could become outdated due to external events such as Kubernetes network policy changes or actions of other users such as adding allowed connections. When moving from one screen to another the new screen is always refreshed with the latest data but if you remain on the same screen, these changes are not automatically reflected.

The current page can however be refreshed manually by clicking on the Refresh icon at the top of the screen at any time.

Sorting Table Columns

Most tables can be sorted in ascending or descending order by one or more columns. Click on the column header of column to sort by the contents of that column. Click again to reverse the sort order.

Resizing Columns

Table column can be resized to make reading easier. Hover over a column divider in the header of a table, and drag the column separator bar right or left.

Filtering

View Filter - Domain, Project and Namespace

Filter by Domain and Project

One domain can be selected at a time. If you have multiple domains, click on the desired domain name to change the current selection.

One project can be selected at a time. If your domain has multiple projects, click on the desired project name to change the current selection.

Filter by Namespace

Data displayed in is aggregated from all namespaces selected. Click on Namespace to change the selection. The filter icon is displayed when not all namespaces are selected. By default, displays a filtered view of the aggregated data in your cluster that includes all application namespaces and excludes system namespaces.

The currently selected filter remains in effect as you navigate to different windows and a notification is displayed in the new connections window.
