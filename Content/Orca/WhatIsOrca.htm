What is ?

Visibility

  • View the pods, services, nodes, Kubernetes network policies and network traffic in your cluster.
  • Identify traffic that is not compliant with your defined policy.

Enforcement

  • Create a security policy for the cluster
  • Build Kubernetes network policies from your policy
  • Prevent access to and from selected domains

Automation

  • Immutable policy control - automated policy creation built into your CI/CD pipeline.
  • Scanning of Docker container images for vulnerabilities and giving an appropriate to each container. computes a for each active service. The gives you instant insight into the security of the services you utilize in your application. This can be added as a step to your CI/CD pipeline.
  • Integration of vulnerability scans into your code repository, implemented as a badge in your readme file. The Tufin badges are an effective way to highlight security issues to the application developers as they develop the application.
  • Application of your policy to firewalls via and directly on Microsoft's Azure Firewall.

Supported Platforms

integrates into any Kubernetes platform, any build server, and any code repository. Your application can be deployed on any public cloud platform that supports Kubernetes, including AWS, Azure, GCP, OpenShift, Docker Enterprise, Minikube, VMware or on your own private on-premises cloud platform.
