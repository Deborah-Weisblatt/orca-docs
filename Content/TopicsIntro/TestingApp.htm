Monitoring Your Application

has now been deployed on your cluster. You can now start monitoring your application.

  1. Log in to . If this is your first time, login by using the Set Password link and user name provided in your Sign Up email.
  2. Go to Services. If the services appear, you're all set! If not, there may be a connectivity issue between your cluster and Orca - please verify that all the ports listed in Outbound Connections are allowed.

  3. The service graph will show traffic as soon as it is generated in the cluster.

    For continuous security, we recommend that you automate this step by running system tests on your application.

  4. If your cluster is hosted on and you want to use the vulnerability scan feature, follow the steps in Docker Image Scanning. This can be done later.

  5. Build your enforcement Policy using the Discovery engine.

  6. You can choose whether to enforce the policy you created by toggling Learning Mode on or off.
