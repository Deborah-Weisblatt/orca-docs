Dashboard

The dashboard gives you a snapshot of your Kubernetes security issues vulnerabilities through charts:

Network Policy Namespace Coverage

This chart compares the number of namespaces having at least one Kubernetes network policy with namespaces that have no network policies. The number of k8s network policies is not reflected here.

Namespace Vulnerabilities

This chart shows the number of namespaces for which container vulnerabilities were found against the highest severity of vulnerability found. The number of vulnerabilities found in each namespace is not reflected here. The vulnerability level for a namespace is the largest vulnerability found in any of its containers.

Privileged Containers

This is a list of containers for which privileged is set to true (privileged=true), therefore giving it access to all devices on the host. The , which is not connected to the container privileged setting, is also shown to provide additional information.
