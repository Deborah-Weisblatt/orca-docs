Securing

Implicit measures that keep secure.

Hosted Environment

The software runs on Google Cloud and uses the Google Security Model, the very same platform used by Google itself to secure its own cloud products. For more information: https://cloud.google.com/security/overview/.

What Data do We Collect?

keeps the following data for the purpose of mapping your network and the traffic within it.

  • k8s namespaces
  • k8s pod and service information including metadata, labels and definitions
  • Connections made to and from nodes in your k8s clusters
  • Node information including metadata, labels and definitions
  • Cloud vendor security definitions (security groups etc)
  • Container images

Encryption

  • All data is encrypted at rest i.e. in database
  • All traffic is encrypted in transit using HTTPS

Vulnerability Checking

  • is vulnerability checked using itself
  • Sanity testing is performed as part of the CI/CD process
  • is tested for all of the OWASP Top 10 application security risks using code review and penetration testing methods.

Multi-Tenancy

  • The database is multi-tenant.

Retention Policy

  • Data collected is retained until the user terminates their account.
