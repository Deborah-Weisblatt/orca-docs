Pods

The Pods window displays all the podsA Kubernetes pod is a group of containers that are deployed together on the same host. in the Kubernetes cluster from nodes that has permission to access:

Column

Description
Name Name of the pod
Namespace Namespace in which the pod is running
Pod Security

If a value appears here, the pod has a permissive security policy, otherwise it will be blank.
Status

Status of the pod:

  • Running
  • Not Running
IP IP address of the pod
Labels All labels defined for the pod
Containers All containers in the pod, showing .

Privileged Containers

When Privileged is set to true for the container (privileged=true), it has access to all devices on the host. Values:

  • Privileged: Container has privileged access
  • Blank: Privileged does not have privileged access
  • View vulnerability details - Click on a to display the vulnerability details for that container.
