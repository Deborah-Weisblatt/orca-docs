Azure Permissions

For to monitor Azure subscriptions follow the step below. For more information on Azure authorizations, see What is role-based access control (RBAC) in Azure

  1. In Azure Active Directory create a new application registration where:

    Name=<Your AppReg name>

    Application Type = Web app / API

  2. For the required subscription, under Access control (IAM), add a role assignment to the application registration created in the previous step where:

    Role=Reader

    Assign access to=Azure AD user, group or service principal

  3. To allow to perform the CIS benchmark audit, you will also need to:

    • Add Azure Active Directory Graph API permission to the application registration
    • Add Directory.read.all permission to the Azure Graph API
    • Grant the application registration admin consent for Default Directory
  4. Add a client secret
