Azure Firewalls

"Azure Firewall is a managed, cloud-based network security service that protects your Azure Virtual Network resources" - official documentation. locates and displays Azure firewalls from all your monitored accounts.

The screen displays a list of all Azure firewalls found in your monitored accounts. It contains the following details:

  • Name - the name of the firewall
  • Account - the name of the account as configured in
  • Region - the geographical region of the firewall
  • Vnet - the network on which the firewall is deployed
  • Public IP - One or more IP addresses at which the firewall can be accessed from the internet
  • Private IP - The IP address of the firewall on the Vnet
  • View firewall details - click on the desired firewall from the list.

View Firewall Details

  1. Click on the desired firewall from the list. The Overview screen appears. When hovering over fields, the copy-to-clipboard icon is displayed to allow easy copy/paste of screen information.

  2. You can switch between the Overview and Rules screens by clicking on Overview or Rules in the side menu.

Rules

The displayed rules are ordered by priority and can be filtered.

The screen is split into NAT rules, network rules and application rules - click on the desired type of rule to switch between them.

The rule details are displayed as specified on the firewall, as follows:

  • Priority
  • Name
  • Source
  • Destination
  • Translated Destination (for NAT rules only)
  • Service
  • Translated Service (for NAT rules only)
  • Action (for network and application rules only)
  • Collection
