This screen lists your applications as specified by the tags associated with your assets. The following information is displayed for each application in the list:

Field
Sequence of item in the list
Application Name in the format specified in the App Manager
The number of assets associated with the application

The cloud vendor
Network connectivity - internet connectivity, network peering and/or on-premise network
Warning - indicates a violation

  • Filter the application list
  • Display security issues for an application
  • Drill down to an application

Filter the Application List

The list can be filtered on the following criteria:

  • Internet connectivity
  • Network peering
  • On-premises connectivity
  • Risks
  • Name (Application name)
  • Tags

Display Security Issues for an Application

If the warning icon is displayed for an application, you can click on it to display the application issues window. The issues will be listed, grouped by type, showing the number of asset involved. Click on View asset list to drill down to the relevant assets.
