Firewalls

locates and displays all and firewalls in your cloud accounts.

The screen displays a list of all firewalls found in your monitored accounts. It contains the following details:

  • Name - the name of the firewall
  • Account - the name of the account as configured in
  • Region - the geographical region of the firewall
  • Vnet - the network on which the firewall is deployed
  • Public IP - One or more IP addresses at which the firewall can be accessed from the internet
  • Private IP - The IP address of the firewall on the Vnet
  • View firewall details - click on the desired firewall from the list.

View Firewall Details

  1. Click on the desired firewall from the list. The Overview screen appears.

  2. (Azure only) You can switch between the Overview and Rules screens by clicking on Overview or Rules in the side menu.

Overview

Account is the account name defined in and all other properties and the tags are as specified for the firewall in your cloud account. When hovering over a property, the copy to clipboard icon appears - click to copy the value.

For Azure

  • Account - the name of the account as configured in

  • Properties of the firewall as configured in your cloud account

    • Name
    • ID
    • Private IP
    • Public IP
    • Vnet
    • Subnet
    • Resource group
    • Region
    • Availability zone
  • Vendor - 'Azure'

For AWS

  • Account - the name of the account as configured in

  • Properties of the firewall as configured in your cloud account

    • Name
    • Description
    • Policy Name
    • Policy Description
    • VPC
    • Region
    • Subnet(s)
    • Availability zone(s)
    • Endpoint ID(s)
  • Vendor - 'AWS'

Rules

For Azure only.

The displayed rules are ordered by priority and can be filtered.

The screen is split into NAT rules, network rules and application rules - click on the desired type of rule to switch between them.

The rule details are displayed as specified on the firewall, as follows:

  • Priority
  • Name
  • Source
  • Destination
  • Translated Destination (for NAT rules only)
  • Service
  • Translated Service (for NAT rules only)
  • Action (for network and application rules only)
  • Collection
