Permissive Access- Applications

The permissive access application view lists all the applications that have one or more assets with overly permissive ingress access.

What is Permissive Access?

The main screen section is a list of applications showing the following information:

  • Application name

  • Internet access:

    Exposed-one or more of the application’s assets has access to the internet

    blank-none of the application’s assets has access to the internet

  • Total assets. The total number of assets in the application

  • Affected assets. The number of assets in the application with permissive access

  • Permissive sources. A list of all source settings that demonstrate permissive access in the application

  • Permissive services. A list of all applications and service settings that demonstrate permissive access in the application

  • Drill down to see the assets for a selected application.

    Click on the desired application.
