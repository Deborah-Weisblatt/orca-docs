Initial Setup

Congrat's! You've signed up for and logged in. The next stage is to get to monitor your public cloud, Kubernetes clusters and on-premise network so you can get visibility into how secure they are and block access that shouldn't be allowed. This is done separately for each environment - cloud, k8s and on-premise:

Every time you log in, the dashboard is the first thing you see; it consists of risk charts for the environments that have been set up and Start buttons for environments not yet set up. Until you set up at least one of these environments, the following screen is displayed:

  • Start monitoring your public cloud by clicking on the public cloud Start button. This will take you to the Account Manager, where you will enter the credentials for all the cloud accounts you want to monitor.
  • Start monitoring your Kubernetes clusters by installing on your clusters.
  • Contact us about monitoring your on-premise network.
