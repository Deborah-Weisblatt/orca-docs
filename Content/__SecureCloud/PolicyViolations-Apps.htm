Policy Violations - Applications

The policy violations application view lists all the applications that have one or more assets in violation of the policy.

What Are Policy Violations?

The main screen section is a list of applications showing the following information:

  • Application name

  • Internet access:

    Exposed-one or more of the application’s assets has access to the internet

    blank-none of the application’s assets has access to the internet

  • Total assets. The total number of assets in the application

  • Affected assets. The number of assets in the application with policy violations

  • Risky ports. A list of all risky ports identified in the application's assets. This is provided as additional information and is not necessarily connected to policy violations.

  • Drill down to see the assets for a selected application.

    Click on the desired application.
