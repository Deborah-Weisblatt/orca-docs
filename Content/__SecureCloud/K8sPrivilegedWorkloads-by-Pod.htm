Privileged Workloads by Pod

This screen displays all pods in a selected namespace with privileged workloads.

The selected cluster and namespace appear at the top of the screen. The main screen section is a list of all pods in the namespace that have privileged workloads, showing:

  • Pod name

  • Host Access. One or more permissive pods security policy values:

  • Labels. One or more label key/values assigned to the pod.

  • Containers. The names of all containers in the pod, together with the their .

  • Privileged mode:

    Privileged - privileged parameter set to true

    blank - privileged parameter set to false
