Privileged Workloads by Namespace

This screen displays all namespaces in a selected cluster with privileged workloads.

The selected cluster's name appears at the top of the screen. The main screen section is a list of all namespaces in the cluster that have privileged workloads, showing:

  • Namespace name

  • Total pods. The number of pods in the cluster

  • Affected pods. The number of pods in the cluster that contain privileged workloads.

  • Host Access. The number of pods with a permissive pod security policy

  • Privileged mode. The number of containers with privileged parameter set to true

  • Drill down to see the privileged workloads by pod.

  • Return to Privileged Workloads - Clusters - click on at the top of the screen
