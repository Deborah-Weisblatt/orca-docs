Kubernetes Vulnerabilities in a Cluster

This screen displays all known vulnerabilities in the Kubernetes implementation on a selected cluster.

The selected cluster's name appears at the top of the screen. The main screen section is a list of the CVE's that apply to the Kubernetes version.

  • Severity. High, Medium, Low, negligible

  • Report. The official CVE code assigned to the vulnerability
  • Fixed in Version. The Kubernetes version, if exists, in which the CVE was fixed
  • Description. A description of the CVE.
  • Return to Kubernetes Vulnerabilities - Clusters - click on at the top of the screen
