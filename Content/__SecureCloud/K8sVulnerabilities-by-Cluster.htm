This screen displays all clusters containing known Kubernetes vulnerabilities.

What are Kubernetes vulnerabilities?

The main screen section is a list of clusters showing:

  • Cluster name

  • Kubernetes Version

  • Platform. The Kubernetes platform e.g. EKS, AKS..

  • Vulnerabilities. The number of vulnerabilities found

  • Severity. The highest severity vulnerability found - Low, Medium, High
