by Namespace

This screen displays all namespaces in a selected cluster with .

The selected cluster's name appears at the top of the screen. The main screen section is a list of all namespaces in the cluster that have , showing:

  • Name. The namespace name

  • Any Allowed. One or more ingress characteristics are allowed for any e.g. ingress to any pod or ingress from any IP address

    Any - ingress to the namespace is allowed to any service due to:

    • The cluster has no network policies for this namespace or
    • The cluster has at least one network policy for this namespace that explicitly allows ingress to all services (spec: podSelector: {} ingress: - {} policyTypes: - Ingress) or
    • The cluster's network policies collectively do not explicitly allow ingress to any specific service (ingress: -from -podSelector: matchLabels:) or

    blank - The cluster has at least one network policy that restricts ingress to the namespace

  • Large Subnets. Ingress to the namespace is allowed from a wide range of IP addresses

    Any - Ingress to the namespace is allowed from a wide range of IP addresses due to:

    • The cluster has no network policies for this namespace or
    • The cluster has at least one network policy for this namespace that explicitly allows ingress from all IP addresses (spec: ingress: - from: - ipBlock: - {} policyTypes: - Ingress) or
    • The cluster's network policies collectively do not explicitly allow ingress from any specific IP address range

    Class A - ingress to the namespace is allowed from any IP address in the subnet 255.0.0.0/8 (or address range 1.0.0.1 to 126.255.255.254)

    Class B - ingress to the namespace is allowed from any IP address in the subnet 255.255.0.0/16 (or address range 128.1.0.1 to 191.255.255.254)

    Class C - ingress to the namespace is allowed from any IP address in the subnet 255.255.255.0/24 (or address range 192.0.1.1 to 223.255.254.254)

    blank - ingress to the namespace is restricted to well-defined port ranges

  • Any Port - ingress to the namespace is allowed from any TCP port or any UDP port

  • Total Services. The total number of services defined in the namespace

  • Any Allowed. The number of services in the namespace that are allowed to access the namespace due to the Any Allowed setting.

  • Large Subnets. The number of services that are allowed to access the namespace due to the Large Subnets setting.

  • Any Port. The number of services that are allowed to access the namespace due to the Any Port setting.
