Permissive Access - Assets

The permissive access application view lists an application's assets that have one or more high-risk ports open.

What is Permissive Access?

The selected application's name appears at the top of the screen. The main screen section is a list of the application's assets showing the following information:

  • Permissive sources. A list of all source settings that demonstrate permissive access in the asset

  • Permissive services. A list of all applications and service settings that demonstrate permissive access in the asset
  • Return to Permissive Access- Applications - click on at the top of the screen
