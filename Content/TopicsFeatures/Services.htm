Services

The services window displays a table of the services in your Kubernetes cluster. The table lists:

Column

Description
Name name of the service
Namespace namespace in which the service is running
Labels all labels defined for the service
Endpoints all endpoints exposed by the service that go through an ingress controller
Ports

the service ports
Public IPs external IP addresses of the service
State (not currently used)

What can I do?

  • Filter - Filter by: namespace
  • Sort - sort on any column displayed

How do I get here?

From the menu in the left panel, click Services.
