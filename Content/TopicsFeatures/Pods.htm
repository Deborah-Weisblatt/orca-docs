Pods

The Pods window displays a table of the podsA Kubernetes pod is a group of containers that are deployed together on the same host. in the Kubernetes cluster. The table displays:

Column

Description
Name Name of the pod severity for this service
Namespace Namespace in which the pod is running
Containers Lists of containers used by the pod, with each title including its
Status Status of the pod. Supported values: Running, Not Running
IP IP address of the pod
Labels All labels defined for the pod

What can I do?

  • View vulnerability details - Click on a to display the vulnerability details for that container.
  • Filter - Filter by: namespace
  • Sort - sort on any column displayed

How do I get here?

From the menu in the left panel, click Pods.
