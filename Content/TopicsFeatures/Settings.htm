Settings

The Settings window displays configuration information and lets you make configuration changes.

What can I do?

  • Manage API tokens
  • Get the Install Agent deployment string needed to deploy
  • Switch between operation modes Learn and Enforce.
  • Set up notifications

API Tokens

You can view, create or delete project-specific API authentication tokens. automatically creates four tokens for each new project. You can create as many additional authentication tokens as you require. The default tokens created for a new project are:

Name Description
badge authentication token to use when integrating in your code repository, for displaying the Tufin badges with your Docker images
CI authentication token to use when integrating in your CI/CD pipeline
deploy authentication token to use when deploying in your cluster
kite authentication token use by the kite pod

Delete a Token

Hover over the line to be deleted; when the appears click on it to delete the token.

Create a New Token

Click Create Token and follow the instructions in Create an authentication token.
