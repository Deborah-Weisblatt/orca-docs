An badge shows the of a single container image in real time. See Badge Concepts.

Implementing the Badge

Inserting badges into repository readme files is a popular and convenient way of making the available in a prominent position. Just enter the API string with the appropriate parameters.

Format:

![Tufin](https://securecloud.tufin.io/api/<>/<>/badges/security-score?image=<pod-image-name>:<image-tag>&token=<view token>)](https://securecloud.tufin.io/ui/#/<>/<>/grid/scans?image=<pod-image name>:<image-tag>)

Where,

Parameter Where to Find the Value
<> Your
<> The monitored
<pod-image-name> Image name as displayed in pod
<image-tag> The tag ID for the appropriate version of the image - should be the last one you created
<view token> > Settings > API Tokens > Token value for [Scope=view Label=badge]

Example:

[![Tufin](https://securecloud.tufin.io/api/My/My/badges/security-score?image=k8s.gcr.io/event-exporter:05&token=6440d59a-97a9-47c8-92a-20963fc47b11)](https://securecloud.tufin.io/ui/#/My/My/grid/scans?image=k8s.gcr.io/event-exporter:05)

Where,

Parameter Value
<> My
<> My
<pod-image-name> k8s.gcr.io/event-exporter
<image-tag> 05
<view token> 6440d59a-97a9-47c8-92a-20963fc47b11
