Notifications and Alerts

Get notifications via a social network when new connections are discovered. A maximum of one message will be sent every hour.

  • Configure a New Connection Alert

Configure a New Connection Alert

  1. Enable / disable the alert on or off using the toggle switch. remembers the URLs you have entered below, even when disabled.

  2. If you want to notify via Slack, when the alert is enabled, enter the Slack URL of an individual or group to receive notifications - see Slack documentation.

  3. If you want to notify via Microsoft Teams, enter the connector URL of an individual or team to receive notifications - see Microsoft documentation.
