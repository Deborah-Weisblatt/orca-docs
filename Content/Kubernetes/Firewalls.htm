Firewalls

allows you to maintain a single security policy and apply it on multiple Kubernetes clusters and multiple firewalls, on premise and in the cloud. Applying the policy can be done in an automated fashion.

You can apply the security policy to firewalls using any of the following methods:

  • On firewalls managed by
  • On one or more Microsoft Azure firewalls. Currently only egress rules are supported.

You can integrate any or all of these methods with .

Each method requires a one-time integration setup. You can then apply your security policy to your firewall devices by using the Apply on firewalls function. This will submit one or more firewall change requests or tickets to the device, the status of which can then be viewed on the Firewall Tickets Screen.
