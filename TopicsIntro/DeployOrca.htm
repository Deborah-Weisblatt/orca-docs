Deploying

To deploy you must run a Tufin-specified bash command from your terminal window, which will add your cluster to . The deployment command line is included in the Sign Up email you received from Tufin.

How do I deploy in my cluster?

  1. Open a terminal.

  2. Point kubectl to your Kubernetes cluster. See Kubernetes documentation for additional details.

  3. Run the bash command string. If this your first time, the command is included in your Sign Up email:

    bash <(curl -s https://orca.tufin.io/api/<domain>/<project>/scripts/deploy-orca -H "Authorization: Bearer <deploy_token>")

The deployment instructions have been verified on standard vanilla Kubernetes deployments. If you are deploying Tufin Orca on a non-vanilla deployment, follow the instructions below for your specific deployment.

  • Google Kubernetes Engine (GKE) or other vanilla Kubernetes deployments - no additional configuration required
